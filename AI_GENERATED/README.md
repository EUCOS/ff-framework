# AI-Generated Helper Material

This directory contains AI-generated helper material for analyzing, profiling, and planning work around the EUCOS free-fermionic framework.

The files under `AI_GENERATED/` are intentionally separated from the human-written framework code. They are notes, maps, and planning documents only. They are not part of the original implementation and should not be treated as reviewed framework source.

The original code must remain untouched unless the research group later chooses to manually integrate reviewed changes. Any future optimization prototypes, benchmarks, or profiling harnesses should be additive and kept separate from the human-written code until they have been reviewed and accepted by the group.
